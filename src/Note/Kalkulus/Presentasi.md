### Nomor 1
**$ f(x) = 3x + 2 $**  
- $ f'(x) = 3 $.  
  Karena turunan konstan positif ($ f'(x) > 0 $), maka $ f(x) $ **selalu meningkat** pada interval seluruh bilangan real $( \mathbb{R} )$.

**Jawaban:**
$ f(x) $ meningkat pada $ (-\infty, \infty) $.

### Nomor 2
**$ h(x) = x^3 - 3x^2 $**  
- $ h'(x) = 3x^2 - 6x = 3x(x - 2) $.  
  Analisis tanda:  
  - $ h'(x) = 0 $ saat $ x = 0 $ atau $ x = 2 $.  
  - Buat tabel tanda untuk interval $ (-\infty, 0) $, $ (0, 2) $, $ (2, \infty) $:  
    - $ h'(x) > 0 $ pada $ (0, 2) $ dan $ (2, \infty) $,  
    - $ h'(x) < 0 $ pada $ (-\infty, 0) $.  

**Jawaban:**
- $ h(x) $ meningkat pada $ (0, 2) \cup (2, \infty) $.  
- $ h(x) $ menurun pada $ (-\infty, 0) $.
